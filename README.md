Front Panel Audio Toggle
=====================

GNOME Shell extension for toggling front panel audio from the shell's audio menu. Useful for cards that do not have automatic front panel sensing (e.g. Asus Xonar DX).

## Manual Installation
    git clone https://github.com/poscar/FrontPanelAudioToggle.git
    mkdir -p ~/.local/share/gnome-shell/extensions/Front\ Panel\ Audio\ Toggle\@oscaraperez.com
    cp -a FrontPanelAudioToggle/* ~/.local/share/gnome-shell/extensions/Front\ Panel\ Audio\ Toggle\@oscaraperez.com/
    rm -rf FrontPanelAudioToggle/
